# ds-4420-project

For the POC model, we chose to start with an AR model of Airbnb rental unit density sourced from Airbnb history in Chicago since the late 2009s over each quarter in an effort to predict rising overall gentrification in the city. The data was sourced from https://insideairbnb.com/get-the-data/. From extracting the dates of reviews on Airbnbs, we are able to estimate the duration that each individual one has been open for. The general trend is increasing vs being stationary, so in future iterations we will apply differencing/ARIMA.
